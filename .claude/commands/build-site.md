# Build Site

Build the static site and show results.

## Instructions

1. Run the build command:
   ```bash
   python3 build.py
   ```
   Or if in a virtual environment:
   ```bash
   source .venv/bin/activate && python build.py
   ```

2. Show the build output

3. List the generated files in `dist/`

4. Report success or any errors

5. Remind the user:
   - To preview locally, open `dist/index.html` in a browser
   - To deploy, commit and push to GitHub
